## About The Project
![Product Name Screen Shot][product-screenshot2]
![Product Name Screen Shot][product-screenshot]

A simple space invaders game made with pygthon. Controls are as following
  - Left player:
      - W "Move forwards"
      - S "Move backwards"
      - A "Move Up"
      - D "Move down"
      - R "Shoot rockets"
      - F "Shoot bullets"
  - Rigth player:
      - Up Arrow "Move forwards"
      - Down Arrow "Move backwards"
      - Left Arrow "Move Up"
      - Rigth Arrow "Move down"
      - RALT "Shoot rockets"
      - RCTRL "Shoot bullets"

## Getting Started

To setup this project locally on your computer follow the steps below.

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/humamchoudhary/space-invaders-with-pygame.git
   ```

- pip
  ```sh
  pip install pygame
  ```
